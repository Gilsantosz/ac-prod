# Checklist PWA e câmera mobile

- [ ] Instalar o PWA em um aparelho Android compatível.
- [ ] Abrir o PWA pelo ícone instalado.
- [ ] Fazer login com um operador de teste.
- [ ] Abrir **Entrada de Produção > Coleta Código/RFID**.
- [ ] Selecionar **Câmera do celular**.
- [ ] Permitir o acesso à câmera.
- [ ] Ler um QR Code real impresso e validar a baixa automática.
- [ ] Ler um código de barras real impresso e validar a baixa automática.
- [ ] Repetir a leitura em ambiente com pouca luz.
- [ ] Acionar a lanterna quando o aparelho oferecer suporte.
- [ ] Negar a câmera e validar o fallback de digitação manual.
- [ ] Desconectar a rede e registrar uma entrada manual.
- [ ] Reconectar e validar a sincronização sem perda de dados.

Registrar aparelho, versão do Android, navegador, resultado e evidências da execução.
